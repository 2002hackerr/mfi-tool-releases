MFI Investigation Tool v3.0
===========================
1. Double-click build.bat to build the EXE (first time only)
2. Run dist/MFI_Investigation_Tool_v3.exe
3. Select Investigation Mode (AUTO or MANUAL)
4. Browse and select:
   - Claims Sheet
   - REBNI Search Result
   - Invoice Search Result
5. Click RUN INVESTIGATION
6. For MANUAL mode: provide SID/Barcode and click CONTINUE for each level.
7. Click SAVE OUTPUT to generate the final Excel report.

MANUAL STEPS:
- In Manual Mode, the tool pauses at each step to allow validation via DICES.
- Barcode/SID for matched rows should be obtained from DICES.
